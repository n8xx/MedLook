# MedLook - AI Health Analysis Platform

MedLook - это платформа для анализа здоровья на основе ИИ, которая анализирует фотографии лица для выявления ранних признаков усталости, стресса, кожных заболеваний и общего состояния здоровья.

## 🚀 Быстрый старт

### Автоматический запуск
1. Установите зависимости Python: `pip install -r ML/requirements.txt`
2. Запустите `start.bat` для автоматического запуска всех сервисов
3. Откройте браузер и перейдите на http://localhost:3000

### Ручной запуск

#### 1. ML Server (Python Flask)
```bash
cd ML
pip install -r requirements.txt
python start_ml_server.py
```
ML сервер будет доступен на http://localhost:5000

#### 2. Бэкенд (Spring Boot)
```bash
cd backend
mvnw.cmd spring-boot:run
```
Бэкенд будет доступен на http://localhost:8080

#### 3. Фронтенд (React)
```bash
cd frontend
npm install
npm run dev
```
Фронтенд будет доступен на http://localhost:3000

## 📋 Функциональность

### ✅ Реализовано
- **Регистрация пользователей** - полная интеграция с бэкендом
- **Загрузка фото** - поддержка drag & drop, валидация файлов
- **Анализ фото с ML** - интеграция с Python ML моделью
- **Результаты анализа** - детальные результаты с симптомами и рекомендациями
- **Управление профилем** - просмотр и удаление загруженных фото
- **Обработка ошибок** - детальная валидация и сообщения об ошибках
- **CORS настройки** - правильная работа фронтенда с бэкендом

### 🔧 API Endpoints

#### Аутентификация
- `POST /api/auth/register` - регистрация пользователя
- `GET /api/auth/test` - тест соединения

#### Управление фото
- `POST /api/users/upload-photo` - загрузка фото пользователя
- `GET /api/users/photo` - получение фото пользователя
- `DELETE /api/users/photo` - удаление фото пользователя

#### Анализ изображений
- `POST /api/analyze` - анализ фото с помощью ML модели
- `GET /api/analyze/health` - проверка состояния ML сервера

#### ML Server (Python Flask)
- `POST /analyze` - анализ изображения (ML сервер)
- `GET /health` - проверка состояния ML сервера

## 🛠 Технологии

### Бэкенд
- **Java 17**
- **Spring Boot 3.x**
- **Spring Data JPA**
- **H2 Database** (в памяти)
- **Maven**

### Фронтенд
- **React 18**
- **Vite**
- **CSS3** с градиентами и анимациями

### ML Server
- **Python 3.8+**
- **Flask** - веб-фреймворк
- **OpenCV** - обработка изображений
- **MediaPipe** - анализ лица
- **scikit-learn** - машинное обучение
- **pandas, numpy** - обработка данных

## 📱 Использование

1. **Регистрация**: Создайте аккаунт с email, паролем и личными данными
2. **Загрузка фото**: Выберите или перетащите изображение для анализа
3. **Анализ фото**: Нажмите "Analyze Photo" для запуска ML анализа
4. **Результаты**: Просматривайте детальные результаты с симптомами и рекомендациями
5. **Управление профилем**: Просматривайте и управляйте загруженными фото
6. **История анализов**: Отслеживайте предыдущие результаты анализа
